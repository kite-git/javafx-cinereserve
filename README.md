CineReserve is a Java and JavaFX-based with an OOP architecture movie ticket booking system designed to modernize and automate the movie ticket purchasing workflow. 
This served as a final output for the following course: LBYCPEI during Term 3 A.Y. 2024-2025. 

CineReserve has features such as:
- Login for viewers and admins
- Book tickets for specific movies
- Visual seat selection with real-time availability
- Complete information about movie is shown such as: title, genre, duration, description, movie poster, and available show times.
- Payment window
- Receipt generation through text file

Technical Requirements:
- Java (SDK 17+)
- JavaFX SDK
- File I/O for receipt generation
- External CSS for good styling

This project demonstrates OOP design, JavaFX integration, MVC structure, and real-world software engineering practices.
